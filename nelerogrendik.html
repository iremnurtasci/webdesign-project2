Neler Öğrendik?

BBY262 Web tasarımı dersinde, web siteleri oluşturmak için HTML ve CSS dillerini kullanmayı öğrendik. Bu ders, öğrencilerin temel web tasarımı kavramlarını anlamalarına yardımcı olur. Dersin ilk haftalarında, HTML'in temellerini öğrenerek, web sayfalarını nasıl yapılandırabileceğimizi öğrendik. Bu aşamada, HTML etiketleri, görüntüler, linkler gibi çeşitli öğelerle ilgili temel bilgileri öğrendik. Daha sonra CSS'e geçerek, web sayfalarımızın tek bir sayfa üzerinden nasıl kontrol edebileceğimizi öğrendik. Bu aşamada, renkler, yazı tipleri, boyutlar, arka planlar ve düzenler gibi temel CSS özelliklerini öğrendik.

Dersin ilerleyen haftalarında, daha karmaşık web sayfaları oluşturmak için HTML ve CSS kullanımını derinleştireceğiz. Web tasarımı, bugünün dünyasında çok önemlidir. Çünkü insanlar artık hemen hemen her şeyi internet üzerinden yapmaktadır. HTML ve CSS dillerini öğrenerek, web sayfalarınızı daha profesyonel ve etkileyici hale getirebiliriz. Web tasarımı dersi, web sayfaları oluşturma konusunda temel bilgileri öğrenmek isteyen herkes için harika bir fırsattır. Bu dersi alarak, HTML ve CSS gibi temel web teknolojilerini öğrenerek, çeşitli web siteleri oluşturabilirsiniz ve bu sayede bu teknolojileri kullanarak daha iyi bir web sitesi oluşturma konusunda özgüven kazanabilirsiniz.
